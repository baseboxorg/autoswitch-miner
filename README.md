# autoswitch-miner

### Installation

Autoswitch-miner requires [Node.js](https://nodejs.org/) v4+ to run.

```sh
$ git clone https://github.com/felixbrucker/autoswitch-miner
$ cd autoswitch-miner
$ npm install
```

### Run

```sh
$ npm start
```
